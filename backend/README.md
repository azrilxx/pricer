# NSC Pricer Backend

Planned stack:
- Node.js + TypeScript + Express
- PostgreSQL (Supabase)

The backend will expose REST APIs for:
- Clients, projects, RFQs and RFQ items
- Pricing runs
- Agreement drafts

In later steps we will add:
- Database migrations
- Services for pricing and agreements
- Integration points for AI and n8n automations
